# 📘API Integrated React App
This is my First React application For Practise only.
<br>
A simple React application that fetches data from an API and displays post content dynamically using reusable components and Tailwind CSS for styling.

## 🚀Features
- Built with React(Vite).
- Uses Tailwind CSS (no separate CSS files).
- Fetches data from a public API.
- Displays content in a Card form.
- Button to navigate through Different content

## 🛠️Tech Stack
* React
* Tailwind CSS
* JavaScript
* API used(JSON API): https://jsonplaceholder.typicode.com/posts
